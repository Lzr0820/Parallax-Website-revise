視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種網頁設計技術，讓背景的移動速度比前景更慢。當訪客向下捲動時，就會產生 3D 的視覺效果，增加深度感並打造更沉浸的瀏覽體驗。視差的概念源自光學錯覺， 因為人眼會把近處的物體視為比遠處更大，於是我們會感覺遠處物體移動較慢。這種錯覺 長期被應用在不同媒體中，營造更真實的效果。最早可追溯到傳統動畫，例如迪士尼的 《白雪公主與七個小矮人》，以及《超級瑪利歐》等遊戲。 隨著 CSS 與 HTML 的進步，視差效果後來演變為今日常見的網頁設計手法。

2007 年，網頁開發者 Glutnix 在部落格分享了瀏覽器中的視差捲動實作，包含 JavaScript 與 CSS 2 的範例與示範，支援當時的 Internet Explorer 6 等瀏覽器。 直到 2011 年 HTML5 與 CSS3 推出後，這項技術才在網頁設計圈迅速普及。支持者認為， 視差能更自然地呈現網頁的流動性；推廣者則視其為提升互動與整體體驗的工具。不過， 普渡大學在 2013 年發表的研究指出：「…雖然視差捲動提升了某些使用者體驗面向，但 未必能提升整體體驗。」


為什麼要使用視差圖像？

    設計師嘗試視差圖像主要有兩個原因：

  • 有助於敘事

    視差圖像非常適合說故事。好的設計重在溝通；當我們設計網站時， 目標是與訪客對話，傳達產品、服務或品牌的精彩故事。視差捲動能 讓訪客沉浸在獨特的線上體驗中，甚至可打造單頁式網站，讓訪客 不需切換頁面就能閱讀完整故事。

  • 提升使用者參與度

    視差圖像能提升使用者參與度。降低網站跳出率是網頁設計的重要目標之一。 跳出率指訪客只看一頁就按下返回或關閉分頁的比例。高跳出率會影響 搜尋排名，因此任何能降低跳出率的做法都值得嘗試。精心設計的視差 頁面可提供動態體驗，讓訪客更願意停留。


若想在網站加入視差捲動效果，請留意以下建議：

  • 適度使用

    視差捲動在特定元素上使用會更有力，而不是全站到處使用。 可用在頁首、標題或首頁即可。目標是強化網站設計，而非分散注意力。

  • 影像壓縮

    視差捲動會使用多種媒體與 CSS 形狀來營造深度感。 請避免因為大圖檔載入過慢而影響效能與使用體驗。

  • 善用色彩

    視差捲動不只是前景與背景圖片，也包含色彩搭配。 可以讓背景使用較低飽和度的色調，前景使用較鮮明的色彩， 以強化深度感。
